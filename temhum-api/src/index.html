Welcome to the IoT Temperature and Humidity API

This API provides endpoints to retrieve and manipulate temperature and humidity data.

Available endpoints:

Endpoint Description
GET /api/data?type=sensors Get all sensors
GET /api/data?type=sensorTypes Get all sensor types
GET /api/data?type=sensorLocations Get all sensor locations
GET /api/data?type=sensorData Get all sensor data
GET /api/data?type=temperature Get temperature readings
GET /api/data?type=high-temperature Get high temperature readings
GET /api/data?type=record&id={recordId} Get a specific record by ID
PUT /api/sensorData/update/{recordId} Update a record by ID
DELETE /api/sensorData/delete/{recordId} Delete a record by ID
POST /api/sensors/insert Insert a new sensor
POST /api/sensorTypes/insert Insert a new sensor type
POST /api/sensorLocations/insert Insert a new sensor location
POST /api/sensorData/insert Insert a new sensor data
